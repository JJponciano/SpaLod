BKG Linked Data and Semantic Data management

Welcome to BKG Linked Data and Semantic Data management.

This Website is the result of the two years project of research and development, entitled “Intelligent data collection, storage and provision within public administration”. This project has been realized in partnership with the institute i3mainz.

It provides the ability to integrate data into a triplestore, verify and modify data, manage related schemas, and explore Linked Data to enrich the data.

Data are available and accessible through a Semantic Web Feature Service following OGC standard. It also provides a metadata catalogue, where you can integrate into the triplestore, visualize and modify metadata. The metadata catalogue is accessible through a web service following the OGC API Records standard.
It gives an access to the triplestore, where data and metadata are semantically stored through RDF graph. Finally, you can create thematic map from integrated Feature collection and Linked Open Data.

Thematic map

Create a thematic map according to a chosen class from Wikidata or from the local ontology.

Enrichment

Enrich the local ontology thanks to a SPARQL query from Wikidata.

Metadata Catalogue

Overview of Metadata to choose and access on the wanted data.
